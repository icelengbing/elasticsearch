[[indices-exists]]
== 索引存在检查

用于检查索引是否存在。例如：

[source,js]
--------------------------------------------------
HEAD twitter
--------------------------------------------------
// CONSOLE
// TEST[setup:twitter]

HTTP 状态代码表明指定索引是否存在。`404`  表示不存在， `200` 表示存在。

IMPORTANT: 此请求不区分索引和别名，即如果存在具有该名称的别名，也会返回状态代码200。
